# 群聊聊天室
# 功能类似QQ微信群聊
#  1 进入聊天室要输入姓名　姓名不能重复
#  2 进入聊天室会向其他人发通知
#  3 某人发消息群里其他人都收到
#  4 某人退出聊天室也向其他人发通知　xxx　退出聊天室
#  5 管理员发送消息　群里人都可以收到
#  chat_server
#（根据功能想技术点）
# 确定技术模型
#    1.消息发送：转发　先发送给服务器然后转发给其他人
#    2.套接字：udp套接字
#    3.存储用户：字典或者列表
#    4.消息收发随意：创建两个进程，收发分开
#注意事项
#    1.封装　可以封装成函数，也可以封装成类
#    2.测试  实现一个功能，测试一个功能
#    3.流程　先搭建网络链接，后实现功能（比如通信项目　首先现将通信建成，才可以编程）
# 　　4.数据库　先设计数据库再实现功能  
#具体功能
#    1.网络通信
#      　服务端：创建UDP套接字，绑定地址　创建多进程（收发分开）
#      　客户端：创建UDP套接字，创建多进进程（收发分开）
#    2.用户登录
#      　服务端：接受姓名
#             判断姓名是否存在
#             根据判断结果回复客户端
#             不允许登录则工作结束　否则允许登录将用户加入数据库
#             将用户登录提示发送给其他人
#      　客户端：输入姓名　将姓名发送给服务器　
#                     接受服务器反馈
#                    （不允许登录重新输入　允许登录则进入聊天室）
#                    创建进程，用于消息发送
#             
#    3.聊天
#         服务端：　接受消息　将消息转发他人
#         客户端　　输入消息　将消息发给服务器
#    4.用户退出
＃　　　　　服务端：收到＃＃表示客户端退出
#                将用户从user中移除
#                告知其他人退出，给该用户发指令退出
#         客户端：　输入＃＃表示退出
＃　　　　　　　　　　将退出信息发给服务器　　
＃　　　　　　　　　　进程退出
＃　　　　　　　　　　接收进程收到服务器指令退出
#
